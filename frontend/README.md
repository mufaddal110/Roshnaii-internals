# Roshnaii Frontend

React frontend for Roshnaii poetry platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update API endpoint in `src/api.js` if needed

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Tech Stack

- React 19
- Redux Toolkit
- React Router
- Vite
- Lucide React (icons)
