# Roshnaii Admin Panel

Admin panel for managing Roshnaii poetry platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the admin panel:
```bash
npm run dev
```

## Features

- Manage Poetry (publish/unpublish/delete)
- Manage Poets (publish/unpublish/delete)
- Admin authentication

## Access

- URL: `/admin/login`
- Only users with `isAdmin: true` can access
